# Assets

Shared visual and 3D assets live here.

- `models/source/`: editable source files from modeling tools, for example Blender `.blend` files.
- `models/exported/`: runtime-ready exports, for example `.glb` or `.gltf` files.

Large binary assets should be discussed before committing. Prefer optimized runtime exports in the
frontend and keep raw source files organized by scene or equipment type.
