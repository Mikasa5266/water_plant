# 未来水厂智能运营系统 Demo

## 运行方式

推荐使用本地静态服务打开，避免浏览器对脚本或资源路径做额外限制。

### 方式一：Python

在目录 `E:\迎风聚智\组内项目\20260525\future-water-plant-demo` 下执行：

```powershell
python -m http.server 9000
```

然后在 Chrome 中打开：

`http://127.0.0.1:9000`

### 方式二：直接双击

可直接打开 [index.html](/E:/迎风聚智/组内项目/20260525/future-water-plant-demo/index.html)，
如果浏览器对本地脚本有限制，再改用方式一。

## 当前版本能力

- 左侧工艺链路切换与演示控制
- 中间 Hightopo 3D 水厂占位场景
- 右侧智能体状态、AI 决策链、执行方案、事件日志
- 底部流程时间轴
- 三条完整演示链路：加药、超滤、膜处理
- 高危链路确认逻辑：超滤、膜处理
