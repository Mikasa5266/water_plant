## 协作接口

A 写入 store → B 从 store 读取驱动 3D：

| A 写入 | B 消费 |
|--------|--------|
| `scenarioStore.phase` | 3D 动画切换 |
| `scenarioStore.particleIntent` | 粒子颜色/方向 |
| `scenarioStore.cameraFocus` | 摄像机聚焦 |
| `scenarioStore.deviceFlashing` | 设备闪烁 |
| `scenarioStore.activeAgent` | Agent 球体高亮 |
| `scenarioStore.thinkingContent` | 思考气泡内容 |

**原则：A 不改 `simulation3d/`，B 不改 `components/`。**
